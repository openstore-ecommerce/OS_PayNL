Open Store Payment Gateway for pay.nl
=====================================

Register new pay.nl account: https://www.pay.nl/registreren/?id=M-9652-6170

Contact us for any help with this plugin: info@Site4All.nl

Important configure
    Set the settings and save:
    Backoffice > Admin > OS_PayNL Payment
    Backoffice > Admin > Plugins > OS_PayNL Payment > Edit (Save! even if you haven't changed anything)
